ggps - gps file utilities for garmin connect and garmin devices
===============================================================

Features
--------

- Parse *.gpx, *.kml, and *.tcx files downloaded from Garmin Connect


Quick start
-----------

Installation:

.. code-block:: console

    $ pip install ggps

Use:

.. code-block:: pycon

    >>> import ggps


Source Code
===========

See `ggps at GitHub <https://github.com/cjoakim/ggps>`_ .
